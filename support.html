2 자주 묻는 질문

영수증 스캔이 잘 안 돼요

다음을 확인해 보세요:

  • 밝은 곳에서 촬영하세요 (그림자 X)
  • 영수증을 구김 없이 평평하게 펴주세요
  • 영수증 전체가 화면 안에 들어오도록
  • 카메라와 20~30cm 거리가 적당해요

그래도 인식이 잘 안 되면 영수증 사진과 함께 메일로 문의해 주세요.

소비기한 알림이 오지 않아요

iPhone 알림 설정과 BIFE 앱 내 설정을 모두 확인해 주세요:

1) iPhone 알림 설정

  • 설정 → 알림 → BIFE → "알림 허용" 켜짐
  • 설정 → 집중 모드 → 방해금지 모드가 켜져 있는지 확인

2) BIFE 앱 내 알림 설정

  • BIFE 앱 → 마이 → 알림 설정 → 수신할 알람 설정

모두 정상인데도 알림이 안 오면 앱을 한 번 종료했다가 다시 켜보세요.

레시피 추천이 잘 안 맞아요

BIFE는 현재 보유 중인 재료를 기준으로 일치율을 계산합니다.

  • 냉장고 안 재료가 정확히 등록되어 있는지 확인
  • 소비기한이 지난 재료는 삭제
  • 재료를 더 많이 등록할수록 추천이 정확해져요
장바구니에 담은 재료가 자동으로 사라져요

장바구니에 추가된 재료는 구매 완료 처리되면 자동으로 냉장고로 이동합니다.

혹시 의도하지 않은 동작이라면 영수증 등록 또는 수동 등록 시점을 확인해 보세요.

계정을 삭제하고 싶어요

앱 내에서 마이 → 계정 설정 → 계정 삭제 메뉴를 통해 직접 진행하실 수 있습니다.

계정을 삭제하시면 서버에 보관된 모든 개인정보가 즉시 파기됩니다. 단, 관계 법령에 따라 보존 의무가 있는 일부 정보(거래 기록 등)는 분리 보관 후 보유 기간 경과 시 파기됩니다.

삭제에 어려움이 있으시면 bitesafeteam@gmail.com 으로 문의해 주세요.

3 개인정보처리방침

1. 수집하는 정보

BIFE는 사용자에게 더 나은 서비스를 제공하기 위해 다음 정보를 수집할 수 있습니다:

  • 사용자 입력 데이터: 냉장고 재료 정보, 장보기 리스트
  • 영수증 이미지: 인식 처리 후 즉시 폐기 (서버 저장 X)
  • 앱 사용 통계: 익명화된 크래시 리포트, 기능 사용 빈도

2. 정보 이용 목적

  • 앱의 핵심 기능 제공 (냉장고 관리, 레시피 추천, 알림)
  • 서비스 품질 개선 및 버그 수정
  • 사용자 문의 응대

3. 정보 보관 및 파기

이용자가 입력·생성한 모든 데이터(식재료, 가계부 기록, 가족 그룹 정보 등)는 Supabase(AWS) 클라우드 데이터베이스에 안전하게 저장됩니다. 회원 탈퇴 시 서버에 보관된 개인정보는 즉시 파기되며, 단 「전자상거래법」 등 관계 법령에 따라 보존 의무가 있는 정보(거래 기록 5년, 분쟁 처리 기록 3년 등)는 분리 보관 후 보유 기간 경과 시 파기됩니다.

4. 제3자 제공

BIFE는 사용자의 개인정보를 제3자에게 판매하거나 공유하지 않습니다.

5. 사용자 권리

사용자는 언제든지 다음 권리를 행사할 수 있습니다:

  • 저장된 정보 열람 및 수정 (앱 내에서 직접)
  • 계정 및 정보 삭제 (앱 내 마이 → 계정 설정 → 계정 삭제)
  • 관련 문의: bitesafeteam@gmail.com

6. 변경 사항

본 방침은 법령 변경 또는 서비스 개선에 따라 수정될 수 있으며, 변경 시 앱 내 공지 또는 본 페이지를 통해 안내드립니다.
