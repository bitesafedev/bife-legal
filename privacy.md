# 바이프(BIFE) 개인정보처리방침

**시행일: 2026년 4월 20일**

바이프(이하 "회사")는 이용자의 개인정보를 중요시하며, 「개인정보 보호법」 및 관련 법령을 준수하고 있습니다. 본 개인정보처리방침은 회사가 제공하는 모바일 애플리케이션 "바이프(BIFE)"(이하 "서비스")에서 이용자의 개인정보가 어떻게 수집·이용·관리되는지 안내합니다.

## 1. 개인정보의 수집 및 이용 목적

회사는 다음의 목적으로 개인정보를 수집·이용합니다.

| 목적 | 처리 항목 |
|------|----------|
| 회원 식별 및 계정 관리 | 이메일, 닉네임, 프로필 사진, 소셜 로그인 식별자 |
| 서비스 제공 (냉장고 관리, 레시피 추천, 가계부, 공유 그룹) | 식재료 정보, 가계부 기록, 영수증 이미지 및 분석 결과, 장보기 이력, 가족 그룹 정보 |
| 유료 구독 관리 | 구독 상태, 구독 시작/만료일 (결제 정보는 저장하지 않음) |
| 푸시 알림 발송 (소비기한 임박 등) | 기기 푸시 토큰 |

| 고객 문의 대응 | 이메일, 문의 내용 |

## 2. 수집하는 개인정보 항목

### 2.1 회원가입 시 (소셜 로그인)

- **Google 로그인**: 이메일 주소, 프로필 이름, 프로필 사진 URL, Google 계정 고유 식별자
- **Kakao 로그인**: 이메일 주소, 닉네임, 프로필 사진 URL, Kakao 계정 고유 식별자

### 2.2 서비스 이용 과정에서 수집되는 정보

- 이용자가 직접 입력·업로드하는 데이터: 식재료명, 수량, 구매일/소비기한, 보관구역, 가격, 메모, 장보기 항목, 영수증 사진
- 공유 그룹에 참여한 경우: 그룹명, 그룹 구성원 정보, 그룹 내 공유된 데이터
- 영수증 스캔 기능 이용 시: 영수증 이미지 (AI 분석을 위해 외부 API로 전송)

### 2.3 자동으로 수집되는 정보

- 기기 정보: 기기 모델, OS 버전, 앱 버전
- 광고 식별자 (IDFA/AAID) — 광고 표시 목적 (프로 플랜 이용자는 해당 없음)
- 접속 로그, IP 주소, 쿠키·세션 정보

### 2.4 결제 관련

결제는 Apple App Store / Google Play를 통해 이루어지며, **신용카드 번호, 계좌 정보 등 결제수단 정보는 회사가 직접 수집·저장하지 않습니다**. 회사는 각 플랫폼이 제공하는 구독 상태 정보만 수신하여 저장합니다.

## 3. 개인정보의 보유 및 이용기간

1. 이용자의 개인정보는 **회원 탈퇴 시 즉시 파기**하는 것을 원칙으로 합니다.
2. 다만 관련 법령에 따라 일정 기간 보존이 필요한 경우에는 해당 기간 동안 보관합니다.
   - 전자상거래법상 계약·청약철회 기록: 5년
   - 소비자 불만·분쟁 처리 기록: 3년
   - 서비스 부정 이용 기록: 1년

## 4. 개인정보의 제3자 제공

회사는 이용자의 개인정보를 제3자에게 제공하지 않습니다. 다만 다음의 경우는 예외로 합니다.

1. 이용자가 사전에 동의한 경우
2. 법령에 의하거나 수사 목적으로 적법한 절차에 따라 수사기관의 요청이 있는 경우

※ **소셜 로그인 과정에서 Google/Kakao로부터 이용자의 정보가 회사로 제공되는 것은 "제3자로부터의 수집"이며, 회사가 제3자에게 개인정보를 제공하는 것이 아닙니다.**

## 5. 개인정보 처리 위탁

회사는 서비스 제공을 위해 다음과 같이 개인정보 처리를 위탁하고 있습니다.

| 수탁자 | 위탁 업무 | 처리 항목 | 위치 |
|--------|----------|-----------|------|
| Supabase (Supabase, Inc.) | 데이터베이스 호스팅, 인증, 파일 스토리지 | 서비스 이용 데이터 일체 | 해외(AWS) |
| Anthropic, PBC | AI 기반 영수증 분석 및 레시피 생성 (Claude API) | 영수증 이미지, 식재료 정보 | 미국 |
| Google LLC (AdMob) | 광고 게재 및 성과 측정 | 광고 식별자, 기기 정보 (무료 플랜 한정) | 해외 |
| Apple Inc. / Google LLC | 푸시 알림 발송 (APNs / FCM) | 푸시 토큰 | 해외 |
| Apple Inc. / Google LLC | 인앱결제 처리 | 거래 ID, 구독 상태 | 해외 |

회사는 수탁자가 개인정보 보호 관련 법령을 준수하도록 계약 시 명시하고 관리·감독합니다.

**영수증 이미지 처리에 관한 추가 고지**: 영수증 스캔 기능을 사용할 경우, 이용자가 촬영한 영수증 이미지는 AI 분석을 위해 Anthropic의 API 서버(미국)로 전송됩니다. Anthropic은 해당 데이터를 API 요청 처리 목적으로만 사용하며, 자체 모델 학습에 사용하지 않습니다.

1. 회원 탈퇴 또는 보유 기간이 경과한 경우 지체 없이 해당 개인정보를 파기합니다.
2. 전자적 파일 형태의 개인정보는 복구·재생이 불가능한 방법으로 영구 삭제합니다.
3. 단, 제3조에 명시된 법령상 보존 의무가 있는 정보는 별도 분리 보관 후 기간 경과 시 파기합니다.

1. **접근 통제**: Row Level Security(RLS)를 통한 데이터 접근 권한 관리

3. **저장 데이터 암호화**: 민감 정보 및 인증 토큰 암호화 저장
4. **접속 기록 보관**: 개인정보 접근 기록 저장
5. **보안 패치**: 주기적인 보안 업데이트 적용

## 9. 광고 식별자 및 맞춤형 광고

1. 무료 플랜 이용자에게는 Google AdMob을 통한 광고가 표시되며, 이때 기기의 광고 식별자(IDFA/AAID)가 사용됩니다.
2. 이용자는 기기 설정에서 광고 식별자를 재설정하거나 맞춤형 광고를 제한할 수 있습니다.
   - iOS: 설정 → 개인정보 보호 및 보안 → 추적
   - Android: 설정 → Google → 광고
3. 회사는 광고 개인화를 위해 "비개인화 광고 요청"(requestNonPersonalizedAdsOnly) 옵션을 기본으로 적용합니다.
